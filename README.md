# FileHelper - 文件管理助手

## 项目介绍

FileHelper 是一个功能强大的文件管理工具，主要用于优化视频查重和文件分类，帮助用户更有效地管理本地文件系统。

### 核心功能

- 🔥 **视频查重**：快速识别和管理重复视频文件，支持预览和删除操作
- 📁 **文件分类**：根据自定义规则自动分类文件到不同文件夹

## 快速入门

### 运行环境

- Python 3.7 或更高版本
- PyQt5

### 安装依赖

```bash
# 创建虚拟环境（可选）
python -m venv venv

# 激活虚拟环境
# Windows
env\Scripts\activate
# macOS/Linux
source venv/bin/activate

# 安装依赖
pip install PyQt5
```

### 运行程序

```bash
python app.py
```

## 功能使用指南

### 视频查重

1. **选择目录**：点击「浏览」按钮选择要检查的目录
2. **开始查重**：点击「开始查重」按钮，程序会自动扫描目录中的视频文件
3. **查看结果**：查重完成后，程序会显示所有重复视频组
4. **操作选项**：
   - **预览**：点击「预览」按钮使用系统默认播放器打开视频
   - **删除**：点击「删除」按钮删除重复文件

### 文件分类

1. **选择目录**：点击「浏览」按钮选择要分类的目录
2. **选择规则方案**：
   - **默认规则**：使用内置的文件类型分类规则
   - **自定义规则**：使用用户自定义的分类规则
3. **规则配置**：点击「规则配置」按钮打开规则配置对话框
   - **编辑规则**：修改现有规则
   - **添加规则**：创建新的分类规则
   - **删除规则**：删除不需要的规则
   - **保存配置**：保存自定义规则
4. **开始分类**：点击「开始分类」按钮，程序会根据规则自动分类文件



## 规则配置说明

### 规则类型

FileHelper 支持四种规则类型：

1. **自定义正则表达式**：使用正则表达式匹配文件名
2. **包含**：匹配文件名中包含指定内容的文件
3. **不包含**：匹配文件名中不包含指定内容的文件
4. **后缀名**：根据文件后缀名进行匹配，多个后缀名用逗号分隔

### 默认规则

程序内置了以下默认规则：

| 规则 | 目标文件夹 |
|------|------------|
| 视频文件 (.mp4, .avi, .mov, .wmv, .flv, .mkv, .webm) | 视频 |
| 图片文件 (.jpg, .jpeg, .png, .gif, .bmp) | 图片 |
| 音频文件 (.mp3, .wav, .flac, .ogg, .m4a) | 音频 |
| 文档文件 (.doc, .docx, .pdf, .txt, .md) | 文档 |
| 压缩文件 (.zip, .rar, .7z, .tar, .gz) | 压缩文件 |
| 可执行文件 (.exe, .msi, .bat, .sh) | 可执行文件 |
| 设计文件 (.psd, .ai, .xd) | 设计文件 |
| 代码文件 (.py, .java, .cpp, .js, .html, .css) | 代码文件 |

## 技术实现

### 核心模块

1. **VideoComparator**：视频比较模块，用于计算视频文件哈希值和查找重复视频
2. **FileOrganizer**：文件分类模块，根据规则自动分类文件
3. **FileUtils**：文件工具模块，用于获取文件属性和信息

### 界面实现

使用 PyQt5 构建图形用户界面，包含两个主要标签页：
- 视频查重
- 文件分类

## 常见问题

### Q: 视频查重速度较慢怎么办？
A: 视频查重需要计算文件哈希值，对于大量视频文件可能会比较耗时。建议分批处理，或在空闲时间运行。

### Q: 文件分类后找不到文件怎么办？
A: 文件分类会将文件移动到对应规则的目标文件夹中，请检查对应文件夹。

### Q: 如何添加自定义文件类型规则？
A: 在文件分类标签页中，选择「自定义规则」，然后点击「规则配置」按钮，添加新的规则。

### Q: 程序支持哪些视频格式？
A: 支持常见的视频格式，包括：.mp4, .avi, .mov, .wmv, .flv, .mkv, .webm
